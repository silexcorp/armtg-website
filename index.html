Welcome to ArMTG

ArMTG allows you to play Magic, The Gathering via Augmented Reality, using a set of printed trackers to fetch card images from Gatherer, allowing for use of multiple decks and formats.
ArMTG offers features like sideboarding and shuffling through the deck interface, without the need to physically shuffle the cards.
ArMTG consists of a deck editor overlay for loading and saving multiple decks, and trying out others' decks, using your device's webcam and interchangeable encoded cards.
